# P_Defense

## Quick start

PowerShell (no activation script present):

```
# Install dependencies
C:\Users\mayuu\Desktop\Palantir_Defense\.venv\Scripts\python.exe -m pip install -r requirements.txt

# Run
C:\Users\mayuu\Desktop\Palantir_Defense\.venv\Scripts\python.exe data_sandbox.py
```

If you prefer activating the venv, there is no `Activate.ps1` in `.venv\Scripts`. Using the venv's `python.exe` directly is equivalent.
